# Matteo Mancini - Personal Portfolio Website Source Files

Questi sono i file di codice sorgente utilizzati per costruire e pubblicare il tuo **Sito Web Portfolio Personale**:

- **`index.html`**: Struttura HTML5 con la presentazione del tuo profilo (Data Scientist EQF 7 & AI Engineer), le tue esperienze reali (IPS S.p.A., INSEE, ENSAI, Sapienza), le schede dei 9 progetti GitHub ed i contatti.
- **`style.css`**: Design System Obsidian Dark Mode con Glassmorphism, sfumature glowing e layout responsive per schermi desktop e mobile.
- **`script.js`**: Logica JavaScript per il filtraggio interattivo dei progetti per categoria, l'effetto scrittura automatica nell'Hero, i contatori animati e l'invio diretto delle mail.
